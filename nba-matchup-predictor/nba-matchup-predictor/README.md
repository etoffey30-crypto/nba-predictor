# NBA Matchup Predictor - Advanced AI Edition

A premium, machine-learning-powered NBA analytics dashboard that uses **Bayesian Inference**, **Cooperative Game Theory (SHAP)**, and **Deep Statistical Analysis** to predict matchup outcomes and identify high-value betting opportunities.

## 🚀 Quick Start Guide

### 1. Installation
Ensure you have Python 3.10+ installed. It is highly recommended to use a virtual environment.

```powershell
# Create virtual environment
python -m venv .venv

# Activate virtual environment
.\.venv\Scripts\Activate.ps1

# Install dependencies
pip install -r requirements.txt
```

### 2. Update Historical Data
Fetch the latest box scores and player logs for the current and previous seasons. This is required for the ML model to learn recent team form.

```powershell
python fetch_real_data.py
```

### 3. Fetch Upcoming Matchups
Get the list of games currently scheduled for today from the iSports API.

```powershell
python fetch_upcoming.py
```

### 4. Run the Prediction Engine
This is the core of the system. It will:
- Train a **Random Forest** model on deep statistical features.
- Compute **True Shooting (TS%)** and **Effective FG% (eFG%)**.
- Generate **SHAP Explanations** (Cooperative Game Theory) for every prediction.
- Calculate **Bayesian Uncertainty bounds** for player props.
- Export everything to the visual dashboard.

```powershell
python predictor.py
```

### 5. View the Dashboard
Simply open the `index.html` file in any modern web browser to view the interactive glassmorphism dashboard.

```powershell
start index.html
```

## 🧠 Advanced Features Explained

### 🎯 AI "Best Bet" Recommendation
The dashboard now explicitly highlights the single highest **Expected Value (EV)** play for every game. It evaluates Moneyline, Spread, and Totals, choosing the option where the AI's mathematical confidence is highest.

### 📊 Game Theory (SHAP) Explanations
Instead of a "black box" prediction, the dashboard shows a mathematical breakdown of **why** the AI favored a team. It visualizes the influence of individual metrics like Pace, Defense, and Player Form using SHAP values.

### 📈 Bayesian Player Props
Player stat projections (Points, Rebounds, Assists) are no longer single numbers. They are displayed with **error bounds** (e.g., 25.1 ± 3.4), representing the probabilistic uncertainty based on the player's recent volatility.

## 📁 System Architecture
- `fetch_real_data.py`: Data ingestion from `nba_api`.
- `fetch_upcoming.py`: Real-time schedule updates.
- `predictor.py`: ML training and mathematical inference engine.
- `dashboard.js`: Premium UI logic and data visualization.
- `data.js`: The exported intelligence payload (generated by the predictor).
